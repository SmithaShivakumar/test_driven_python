Temperature

In this lesson you'll build two functions, f_to_c and c_to_f that will convert readings from one scale into another. For this particular example, we don't want to have to deal with decimal precision, so each function should return an integer reading after the conversion.

Fahrenheit and Celsius

One of the key differences between Fahrenheit and Celsius is that the freezing point of water is 0 degrees on the Celsius scale, and 32 degrees on the Fahrenheit scale.

Beyond that, each degree of Fahrenheit is 5/9 of one degree Celsius.

How is a float different than an integer?

floats are a data type in Python that represent numbers. They're different from integers because floats can represent a larger subset of numbers. floats have decimal points. integers will always represent "whole" numbers.

When doing integer math in Python, it will always round down.

Python2 uses integer math for math operations when you input integers.

5 / 3 = 1

However, Python3 uses float math when dividing, so instead you'll get

5 / 3 = 1.6666666666666667

Your code implementation for this challenge will be different based on which version of python you're using. If you aren't sure whether you're using Python2 or Python3 type python --version in your terminal with your virtualenv activated. It will print out your current version.

Lesson objectives

  • Floating point math

Challenge Rules:

  1. f_to_c
    • Accepts a float
    • Converts a Fahrenheit reading to Celsius
    • Returns an integer
  2. c_to_f
    • Accepts a float
    • Converts a Celsius reading to Fahrenheit
    • Returns an integer

Example

freezing point
f_to_c(32) #=> 0
c_to_f(0) #=> 32
boiling point
f_to_c(212) #=> 100
c_to_f(100) #=> 212
