Basic Calculator

In this lesson you'll build a basic calculator script that has the following functions:

  • add accepts two integers and returns the sum of those integers
  • subtract accepts two integers and returns the first minus the second
  • sum accepts a list of integers and returns the sum of all integers in the list.

Lesson objectives

  • Perform basic mathematical operations
  • Learn to interact with list elements
  • Create a python file containing multiple functions

Challenge Rules:

add
  1. The function should accept two integers
  2. The function should return one integer equal to the sum of the two inputs
  3. The function should work for positive and negative integers
subtract
  1. The function should accept two integers
  2. The function should return one integer equal to the first integer minus the second
  3. The function should work for positive and negative integers
sum
  1. The function should accept a list of integers
  2. The function should return one integer equal to the sum of all the list elements
  3. The function should work for a list of any size
  4. The function should work for positive and negative integers
  5. Edge case:
    • If the function receives an empty list, return 0

Example

add(2, 3) #=> 5
add(-2, 3) #=> 1
add(-10, -3) #=> -7

subtract(5, 3) #=> 2
subtract(10, -3) #=> 13
subtract(-12, -2) #=> -10

sum([1, 2, 3]) #=> 6
sum([2, 3]) #=> 5
sum([8, 4, -2]) #=> 10
sum([]) #=> 0
sum([0]) #=> 0
