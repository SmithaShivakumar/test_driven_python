Basic Calculator

In this lesson you'll build a basic calculator script that has the following functions:

  • add accepts two integers and returns the sum of those integers
  • subtract accepts two integers and returns the first minus the second
  • total_sum accepts a list of integers and returns the sum of all integers in the list.

Lesson objectives

  • Perform basic mathematical operations
  • Learn to interact with list elements
  • Create a python file containing multiple functions
  • Read some Python documentation!

Challenge Rules:

add
  1. The function should accept two integers
  2. The function should return one integer equal to the sum of the two inputs
  3. The function should work for positive and negative integers
subtract
  1. The function should accept two integers
  2. The function should return one integer equal to the first integer minus the second
  3. The function should work for positive and negative integers
total_sum
  1. The function should accept a list of integers
  2. The function should return one integer equal to the sum of all the list elements
  3. The function should work for a list of any size
  4. The function should work for positive and negative integers
  5. Edge case:
    • If the function receives an empty list, return 0

Working with iterables!

The total_sum function probably requires some additional Python concepts you might not be familiar with yet. You'll need to learn to move through each element of a list, or iterate over the contents of the list.

In previous lessons, the lesson's index.html file has walked you through everything you need to know in order to accomplish the challenge goals. This time, however, you'll need to take a peek at the Python documentation on for loops.

See if you can find the information you need in the linked documentation. If you still need help, I recommend googling your question (or even googling your error message!). Learning to ask the right questions and hunt for answers is one of the most important skills you can learn as a developer.

Example

add(2, 3) #=> 5
add(-2, 3) #=> 1
add(-10, -3) #=> -7

subtract(5, 3) #=> 2
subtract(10, -3) #=> 13
subtract(-12, -2) #=> -10

total_sum([1, 2, 3]) #=> 6
total_sum([2, 3]) #=> 5
total_sum([8, 4, -2]) #=> 10
total_sum([]) #=> 0
total_sum([0]) #=> 0
