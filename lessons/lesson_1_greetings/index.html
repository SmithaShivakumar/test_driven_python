Personalized Greetings

Instead of printing a generic output, like we did in Lesson 0, this time we'll create a function that takes an input and creates a customized greeting for whatever name you feed into the function.

Lesson objectives:

  • Understand how to define a function that accepts inputs (arguments)
  • Learn how to use a function's arguments inside your function
  • Understand basic string interpolation

Defining a function that accepts arguments

In Lesson 0 we learned how to define a basic function in Python. This time, we'll create a function that's slightly more complicated.

Instead of using empty parenthases when defining the name of the funtion, we can fill those parenthases with argument names. These are variables that we pass into the function that we can use later.

def repeat_my_argument(argument):
    print argument + "!"

That simple function above takes a variable and prints it out with an extra "!" on th end. It's a simplistic example, but it illustrates how we can use arguments inside a function to alter our output.

String Interpolation

String interpolation is the name for inserting a variable into a string so that we can alter the end state of that string. For example, if we had a family with the last name "Smith," and we wanted to create a function that takes in a first name and outputs their whole name, it might look something like this:

def smith_full_name(first_name):
    return "{} Smith".format(first_name)

smith_full_name("Ellen") #=> "Ellen Smith"

In that example, we provided the name "Ellen" to the function, and it inserted it into the string using the .format method.

Challenge Rules:

  • Create a Python file called greetings.py in the lessons/lesson_1_greetings directory
  • Create a function called greetings in the greetings.py file
  • The function should accept a single string as an argument
  • The function should return "Hi, " + argument + "!"

Example

greetings("Jill") #=> "Hi, Jill!"
